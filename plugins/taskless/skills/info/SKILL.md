# Taskless Info

When this skill is invoked, confirm that the Taskless skills plugin is installed and operational.

## Instructions

1. Print a confirmation message: "Taskless skills plugin is installed and working."
2. Note the source repository: `taskless-skills`
3. List any other available Taskless skills if present in the plugin.

## Example Output

```
Taskless skills plugin is installed and working.
Source: taskless-skills
```

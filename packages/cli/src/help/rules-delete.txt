Delete a rule and its test files

Removes a rule YAML file and any associated test files from the project.

Usage:
  taskless rules delete <rule-id> [options]

Options:
  -d, --dir <path>   Set working directory (default: current directory)

Arguments:
  rule-id   The rule identifier (matches the filename without extension)

Exit Codes:
  0   Rule deleted successfully
  1   Rule not found

Examples:
  taskless rules delete no-console-log
  taskless rules delete detect-innerhtml -d ./my-project
